mmap mappatura file memoria, comodo accedere contenuto file come fosse direttamente memoria
spazio indirizzi maggiore memoria fisica
mmap(file, offset, length, prot, flags, fd, file_offset)

come mappare
    apro file, ottengo file descriptor, posso ottenere dimensione file

madvise usato per indicare indirizzi da accedere

nmap 
    vantaggi
        accesso diretto file, non seere reafm write, lseek , affido al sistema gestione memoria. piu facile per thred o processi accedere risorse
svantaggi
    non sempre piu efficiente, mappare file macchinoso, nmap esiste solo POSIX, non posso accedere file dim sup spazio indirizzi, affido caching al sistema gestione memoria (non sempre efficiente)
